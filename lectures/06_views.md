# 1. 뷰 소개

뷰는 하나 이상의 테이블에서 유도된 가상 테이블입니다. 뷰는 실제 데이터를 가지고 있지 않지만, 뷰를 통해 데이터를 볼 수 있습니다.

### 뷰의 장점:
- 복잡한 쿼리를 단순화
- 데이터 보안 강화 (특정 열에만 접근 권한 부여 가능)
- 데이터 독립성 제공

---

# 2. 뷰 생성

기본 구문:
```sql
CREATE VIEW 뷰이름 AS
SELECT 문;
```

### 예제:
```sql
CREATE VIEW 고학년학생 AS
SELECT 학번, 이름, 학과, 학년
FROM 학생
WHERE 학년 >= 3;
```

---

# 3. 뷰 사용

뷰는 일반 테이블처럼 사용할 수 있습니다.

### 예제:
```sql
SELECT * FROM 고학년학생;
```

---

# 4. 뷰 수정

기본 구문:
```sql
CREATE OR REPLACE VIEW 뷰이름 AS
SELECT 문;
```

---

# 5. 뷰 삭제

기본 구문:
```sql
DROP VIEW 뷰이름;
```
